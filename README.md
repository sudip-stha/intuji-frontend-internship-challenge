# Tech Stack

- HTML – Semantic structure
- CSS
- Sass (SCSS) – Modular styling
- Flexbox & CSS Grid – Layout - system
- Media Queries – Responsive design

# Setup

- Sass installed globally
  ( npm install -g sass )
- watch for changes( sass --watch styles/main.scss styles/main.css
  )

# Run the Project

- Open index.html directly in your browser.
